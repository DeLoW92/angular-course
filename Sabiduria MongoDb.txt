Sabiduría MongoDB

1.Como lanzar la base de datos 
cmd -> "C:\Program Files\MongoDB\Server\3.6\bin\mongod.exe" --config C:\Dev\Data.Config\mongod.cfg
-- monogod.cfg es un fichero que he creado previamente

2. Mongod.cfg 
    systemLog:
    destination: file
    path: "C:\\tools\\mongodb\\db\\log\\mongo.log"
    logAppend: true
    storage:
    dbPath: "C:\\tools\\mongodb\\db\\data"
    security:
    authorization: enabled --OJO!! Solo habilitar cuando hayas dado de alta un usuario administrador

3. Añadir un usuario con accesos de administrador:
cmd -> “C:\Program Files\MongoDB\Server\3.6\bin\mongo.exe” -- Abre la consola de comando de tu base de datos.

        use admin
        db.createUser(
        {
            user: "admin",
            pwd: "abc123!",
            roles: [ { role: "root", db: "admin" } ]
        }
        );
        exit;

5. Mongodb trabaja con archivos Bson, el InternalId de mongodb tiene que ser de tipo ObjectId y además tener la etiqueta BsonId
 [BsonId]
        // standard BSonId generated by MongoDb
        public ObjectId InternalId { get; set; }
